Info
ADV berekening

Elke werkdag waarop je 8u werkt (i.p.v. 7,6u) bouwt 0,4u ADV-surplus op. Na 19 werkdagen van 8u heb je 7,6u opgebouwd = 1 ADV-dag.

Overuren

Overuren worden per project berekend. Uren boven het dagelijks doel van het project (of het standaard uren/dag als er geen doel is ingesteld) tellen als overuren.
